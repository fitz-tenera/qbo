Tenera QuickBooks Internal Reporting Tool - Terms of Use

Last updated: June 23, 2026

Internal use only

This internal application is provided by Tenera Care for use only by authorized Tenera Care personnel.

Purpose

The application connects to Tenera Care's own QuickBooks Online company account using Intuit's API and retrieves accounting reports, including Profit and Loss and Balance Sheet reports, for internal management reporting.

Restrictions

The application is not offered to customers, vendors, third parties, or the general public. Unauthorized access or use is prohibited.

User responsibilities

Users must comply with Tenera Care's internal finance, information security, and access-control policies. Data retrieved through this application must be used only for authorized Tenera Care business purposes.

Changes to the tool

Tenera Care may modify, suspend, or discontinue this internal tool at any time.

Contact

Questions about this internal application may be directed to stephen@tenera.care.
